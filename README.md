# AngularApp

## Requirements

- Node.js 18+ and npm
- Angular CLI (optional, you can use `npx`)

## Install

```bash
npm install
```

## Start

```bash
npm run start
```

Open `http://localhost:4200/`.

## Build

```bash
npm run build
```

## Watch

```bash
npm run watch
```

## Test

```bash
npm run test
```
